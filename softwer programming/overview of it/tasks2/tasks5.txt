

## 1. create a simple IPL Fan Bot that takes your favorite IPL team name as input and uses if-else-if statements to print a unique cheer message for each team (e.g., 'Go Mumbai Indians!', 'Chennai Super Kings for the win!'). If the team is not recognized, print 'Team not found!'

```javascript
let team = prompt("Enter your favorite IPL team:");

if (team === "Mumbai Indians") {
    console.log("Go Mumbai Indians!");
}
else if (team === "Chennai Super Kings") {
    console.log("Chennai Super Kings for the win!");
}
else if (team === "Royal Challengers Bengaluru") {
    console.log("Play Bold, RCB!");
}
else if (team === "Kolkata Knight Riders") {
    console.log("Korbo Lorbo Jeetbo Re!");
}
else if (team === "Gujarat Titans") {
    console.log("Go Gujarat Titans!");
}
else {
    console.log("Team not found!");
}
```

##2.Build a Zomato-style food suggestion tool: take the user's preferred meal time ('breakfast', 'lunch', 'dinner', or 'snack') and use a switch-case statement to suggest a popular dish for that time. If the input doesn't match any meal, suggest 'Try some fruits!'.

```javascript
let meal = prompt("Enter meal time (breakfast, lunch, dinner, snack):");

switch (meal) {
    case "breakfast":
        console.log("Suggested Dish: Poha");
        break;

    case "lunch":
        console.log("Suggested Dish: Paneer Biryani");
        break;

    case "dinner":
        console.log("Suggested Dish: Butter Chicken with Naan");
        break;

    case "snack":
        console.log("Suggested Dish: Samosa");
        break;

    default:
        console.log("Try some fruits!");
}
```

## 3.Create a Flipkart discount calculator that asks the user for the total cart amount. Use nested if statements to check: if amount > 2000, apply 20% discount; else if amount > 1000, apply 10% discount; else, no discount. Print the final amount to pay.
     <br><br><em><strong>Hint:</strong> Use nested ifs to check each discount slab.</em>

```javascript
let amount = Number(prompt("Enter Total Cart Amount:"));
let finalAmount;

if (amount > 1000) {

    if (amount > 2000) {
        finalAmount = amount - (amount * 20 / 100);
        console.log("20% Discount Applied");
    }
    else {
        finalAmount = amount - (amount * 10 / 100);
        console.log("10% Discount Applied");
    }

}
else {
    finalAmount = amount;
    console.log("No Discount Applied");
}

console.log("Final Amount to Pay: ₹" + finalAmount);
```

## 4.Write a program that takes a user's age and checks eligibility for three things using if-else statements: if age >= 18, print 'Eligible for Driving License'; if age >= 21, also print 'Eligible for Credit Card'; if age >= 25, also print 'Eligible for Car Rental'. Print all applicable messages for the given age.

```javascript
let age = Number(prompt("Enter Your Age:"));

if (age >= 18) {
    console.log("Eligible for Driving License");

    if (age >= 21) {
        console.log("Eligible for Credit Card");
    }

    if (age >= 25) {
        console.log("Eligible for Car Rental");
    }
}
else {
    console.log("Not Eligible for Driving License");
}
```

### Example Output (Age = 26)

```text
Eligible for Driving License
Eligible for Credit Card
Eligible for Car Rental
```

### Example Output (Age = 19)

```text
Eligible for Driving License
```

### Example Output (Age = 16)

```text
Not Eligible for Driving License
```
