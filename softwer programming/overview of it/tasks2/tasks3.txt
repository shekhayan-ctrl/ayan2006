

1.Declare variables for a Flipkart product: productName (as a string), price (float), and rating (double). 
  Assign sample values and print each variable with its data type.

```c
#include <stdio.h>

int main()
{
    char productName[] = "Wireless Earbuds";
    float price = 1499.99;
    double rating = 4.6;

    printf("Product Name (String): %s\n", productName);
    printf("Price (Float): %.2f\n", price);
    printf("Rating (Double): %.1lf\n", rating);

    return 0;
}
```

**Output:**

```text
Product Name (String): Wireless Earbuds
Price (Float): 1499.99
Rating (Double): 4.6
```

2. Create a constant variable to store the GST rate (for example, 18%) and use it to calculate the final price of a Zomato order with a given base price.
   <br><br><em><strong>Constraint:</strong> The GST rate must not be changeable after its initial assignment.</em>

```c
#include <stdio.h>

int main()
{
    const float GST = 18.0;
    float basePrice = 500.0;
    float finalPrice;

    finalPrice = basePrice + (basePrice * GST / 100);

    printf("Base Price = %.2f\n", basePrice);
    printf("GST = %.2f%%\n", GST);
    printf("Final Price = %.2f\n", finalPrice);

    return 0;
}
```

**Output:**

```text
Base Price = 500.00
GST = 18.00%
Final Price = 590.00
```

## 3.Write a program that stores your favorite Spotify playlist's name (string), total number of songs (int), and average song duration in minutes (float). Print all values in a single formatted sentence.

```c
#include <stdio.h>

int main()
{
    char playlistName[] = "Top Hits 2026";
    int totalSongs = 25;
    float averageDuration = 3.8;

    printf("My favorite Spotify playlist is %s, it has %d songs and the average song duration is %.1f minutes.\n",
           playlistName, totalSongs, averageDuration);

    return 0;
}
```

**Output:**

```text
My favorite Spotify playlist is Top Hits 2026, it has 25 songs and the average song duration is 3.8 minutes.
```
