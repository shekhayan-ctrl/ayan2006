## 1.Given the following buggy code meant to calculate the total price of a Zomato order, identify and fix the syntax and runtime errors:<br><br>let items = ["Burger", "Pizza", "Fries"];<br>let prices = [120, 250, 90];<br>let total = 0;<br>for (i = 0; i < items.length; i++) {<br> total =+ prices[i]<br>}<br>console.log("Total price is: " + total);<br><br><em><strong>Hint:</strong> Watch for assignment and loop variable issues.</em>

let items = ["Burger", "Pizza", "Fries"];
let prices = [120, 250, 90];
let total = 0;

// Loop through all items
for (let i = 0; i < items.length; i++) {
    total += prices[i];   // Add each price to total
}

console.log("Total price is: " + total);
```

### Output

```text
Total price is: 460
```

### Errors Fixed

* Declared the loop variable using `let i`.
* Changed `total =+ prices[i]` to `total += prices[i]` (addition assignment).

## 3.Write a function formatFollowersCount(count) that takes a number and returns a formatted string like Instagram: 1500 as '1.5K', 1200000 as '1.2M', and numbers below 1000 as-is. Add clear comments and use proper indentation.


```javascript
// Function to format follower count like Instagram
function formatFollowersCount(count) {

    // Convert to millions
    if (count >= 1000000) {
        return (count / 1000000).toFixed(1) + "M";
    }

    // Convert to thousands
    else if (count >= 1000) {
        return (count / 1000).toFixed(1) + "K";
    }

    // Numbers below 1000 remain unchanged
    else {
        return count.toString();
    }
}

// Examples
console.log(formatFollowersCount(1500));
console.log(formatFollowersCount(1200000));
console.log(formatFollowersCount(950));
```

### Output

```text
1.5K
1.2M
950
```

## 4.You are given a piece of JavaScript code that should print all even numbers from 1 to 10, but it doesn't work as expected:<br><br>for (let i = 1; i <= 10; i++) {<br> if (i % 2 = 0) {<br> console.log(i)<br> }<br>}<br><br>Find and fix the error, then rewrite the code with good indentation and at least one comment explaining the logic.


### Correct Code

```javascript
// Print all even numbers from 1 to 10
for (let i = 1; i <= 10; i++) {

    // Check whether the number is even
    if (i % 2 == 0) {
        console.log(i);
    }

}
```

### Output

```text
2
4
6
8
10
```

### Error Fixed

The original code used the assignment operator (`=`):

```javascript
if (i % 2 = 0)
```

It should use the comparison operator (`==` or `===`):

```javascript
if (i % 2 == 0)
```
