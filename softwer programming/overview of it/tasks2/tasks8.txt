## 1.Declare a function called getUserInitials that takes a user's full name (like 'Virat Kohli') and returns their initials in uppercase (e.g., 'VK'). Call this function with your favorite cricketer's name and print the result.

function getUserInitials(fullName) {
    let words = fullName.trim().split(" ");
    let initials = "";

    for (let word of words) {
        initials += word.charAt(0).toUpperCase();
    }

    return initials;
}

// Example
let result = getUserInitials("Virat Kohli");
console.log(result);
```

**Output**

```text
VK
```

##2.Create a function addToCart that takes a shopping cart array and a product name, adds the product to the cart, and prints the updated cart. Demonstrate how passing the cart array by reference allows changes to persist outside the function.<br><br><em><strong>Hint:</strong> In languages like JavaScript, arrays are passed by reference. In C/C++, use pointers for reference behavior.</em>

function addToCart(cart, productName) {
    cart.push(productName);
    console.log("Updated Cart:", cart);
}

let cart = ["Laptop", "Mouse"];

addToCart(cart, "Keyboard");

console.log("Cart Outside Function:", cart);
```

**Output**

```text
Updated Cart: [ 'Laptop', 'Mouse', 'Keyboard' ]
Cart Outside Function: [ 'Laptop', 'Mouse', 'Keyboard' ]
```

**Explanation:**
Arrays in JavaScript are passed by **reference**, so changes made inside the function are reflected outside the function.

##3.Write two functions: increaseFollowersByValue and increaseFollowersByReference. Each should take a followers count and add 1000 to it, but one should use pass-by-value and the other pass-by-reference. Show how the original followers count changes (or doesn't) after calling each function.


```javascript
// Pass by Value
function increaseFollowersByValue(followers) {
    followers += 1000;
    console.log("Inside Value Function:", followers);
}

// Pass by Reference (Object)
function increaseFollowersByReference(user) {
    user.followers += 1000;
    console.log("Inside Reference Function:", user.followers);
}

let followers = 5000;

increaseFollowersByValue(followers);
console.log("Outside Value Function:", followers);

let instagramUser = {
    followers: 5000
};

increaseFollowersByReference(instagramUser);
console.log("Outside Reference Function:", instagramUser.followers);
```

**Output**

```text
Inside Value Function: 6000
Outside Value Function: 5000

Inside Reference Function: 6000
Outside Reference Function: 6000
```

##4.Build a function formatPrice that takes a price in rupees and returns a string formatted like Flipkart's price tag (e.g., '₹1,599'). Use this function to display prices for three different products.


function formatPrice(price) {
    return "₹" + price.toLocaleString("en-IN");
}

console.log("Laptop:", formatPrice(55999));
console.log("Smartphone:", formatPrice(15999));
console.log("Headphones:", formatPrice(2499));
```

**Output**

```text
Laptop: ₹55,999
Smartphone: ₹15,999
Headphones: ₹2,499
```

## 5.Refactor an existing function you wrote above to make it reusable for both product names and usernames (for example, a function that capitalizes the first letter of any string).<br><br><em><strong>Constraint:</strong> The refactored function should work for any string input, not just a specific use case.</em>


```javascript
function capitalizeFirstLetter(text) {
    if (text.length === 0) {
        return "";
    }

    return text.charAt(0).toUpperCase() + text.slice(1);
}

// Product Name
console.log(capitalizeFirstLetter("laptop"));

// Username
console.log(capitalizeFirstLetter("virat18"));
```

**Output**

```text
Laptop
Virat18
```

**Explanation:**
The `capitalizeFirstLetter()` function is reusable because it works with **any string**, such as product names, usernames, city names, or any other text.
